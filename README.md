# Sud Dev - Personal Website

## Introduction
Sud Dev is a personal website owned by me, containing information about myself, portfolio projects, CV, certificates, as well as contact details to reach me.

## Features
* Functional contact form
* Project previews

## Tech Stack

* Typescript
* Tailwind CSS
* MongoDB
